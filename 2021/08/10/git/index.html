git

  • git init 初始化仓库
  • git签名（与远程仓库无关）：
    • git config user.name test
    • git config user.email test 项目级别（./git/config）
    • git config –global user.name test
    • git config –global user.email test@jsljd.com 系统级别（c:/user/.gitconfig）
  • git常用命令
    • git add
    • git commit -m “message” filename
    • git status
    • git log
    • git log –pretty=oneline
    • git log –oneline（只有过去版本）
    • git reflog(从一个版本到另一个版本需要的步数,所有版本都能展示)
    • git reset –hard
  • git版本回退
    • 基于索引（git reset –hard 索引值） git reflog获得索引 git reset –hard 索引
    • ^符号（只能后退版本，一个^一个版本）git reset –hard ^
    • ~符号 git reset –hard ~3
    • reset 三个参数对比
      • soft 仅在本地库移动指针
      • mixed 在本地库移动制针并重置暂存区
      • hard 在本地库移动制针并重置暂存区和工作区
  • git找回删除文件
    • 前提是文件已经提交到本地库
    • 删除操作已经提交到本地库 git reset –hard ^
    • 删除操作只提交到暂存区 git reset –head head
  • 比较文件
    • git diff 【文件名】（工作区和暂存区比较）文件名可省，比较所有变化
    • git diff 历史版本号 【文件名】（工作区和本地库比较）
